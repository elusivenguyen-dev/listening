## 📋 How to Use This Assessment Tool

### For Individual Learners
1. **Rate yourself** on each capability using the 12-tier scale
2. **Be honest** — this is for your growth, not for performance review
3. **Focus on behaviors** — what you actually do, not what you know
4. **Review quarterly** — capabilities evolve with practice
5. **Track progress** — use the progress tracker at the end

### For Teams
1. **Each member self-assesses** independently
2. **Aggregate** team scores to identify collective gaps
3. **Design learning paths** based on team needs
4. **Reassess** every 6 months

### For Organizations
1. **Sample leadership population** across levels
2. **Map capability coverage** against strategic priorities
3. **Identify critical gaps** at enterprise level
4. **Invest** in capability building where gaps are largest
5. **Measure ROI** through reassessment

## 🎯 Assessment Framework

### The 12-Tier Rating Scale

| Tier | Name | Self-Description | Behavioral Evidence |
|------|------|-----------------|---------------------|
| **0** | Unaware | "I didn't know this existed" | Never encountered |
| **1** | Foundation | "I've heard of this" | Can define basic terms |
| **2** | Operational | "I can use basic tools" | Applied tools in simple cases |
| **3** | Tactical | "I can execute in my domain" | Led projects using this capability |
| **4** | Strategic | "I can integrate across domains" | Designed cross-functional solutions |
| **5** | Advanced | "I can analyze causally" | Built models, ran experiments |
| **6** | Expert | "I use AI/advanced methods" | Automated or augmented with AI |
| **7** | Master | "I operate at network scale" | Influenced ecosystem-level outcomes |
| **8** | Architect | "I design interventions" | Created frameworks used by others |
| **9** | Visionary | "I predict and generate" | Anticipated trends, created new paradigms |
| **10** | Transformative | "I self-organize" | Built self-sustaining systems |
| **11** | Transcendent | "I see the philosophy" | Wrote/thought at meta-level |
| **12** | Omega | "I am the system" | Embodied the capability completely |

### Scoring Instructions

For each capability, rate yourself **0–12** based on:
- **Knowledge:** Do you understand the concepts? (Tiers 1–3)
- **Skill:** Can you apply the tools? (Tiers 2–5)
- **Integration:** Can you combine with other capabilities? (Tiers 4–7)
- **Creation:** Can you design new approaches? (Tiers 7–10)
- **Being:** Is this part of who you are? (Tiers 10–12)

**Use the average** of these 5 dimensions, rounded to nearest integer.
