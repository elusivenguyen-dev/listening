## Master Formula — Financial Intelligence Effectiveness (FIE)

```
FIE = (Return × Risk_Adjusted_Return × Stewardship × Transparency) / (Greed × Fear × Short_Termism)

Where:
- Return = Absolute performance (0-10)
- Risk_Adjusted_Return = Sharpe, Sortino, alpha (0-10)
- Stewardship = Long-term capital preservation (0-10)
- Transparency = Openness about finances (0-10)
- Greed = Excessive risk-taking for return (inverse, 0-10)
- Fear = Panic-driven decisions (inverse, 0-10)
- Short_Termism = Quarterly focus over decadal (inverse, 0-10)

Target FIE: Personal > 5.0 | Manager > 7.0 | Executive > 8.5 | CEO > 9.0
```
