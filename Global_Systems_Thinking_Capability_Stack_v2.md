### 7 Integration Pipelines (cập nhật)

1. **Research Production:** Read Paper → Map → Write → Present → Decide → Systematize
2. **Strategic Decision:** Read 10-K → Analyze → Write Memo → Board Present → Decide → Model System

## Master Formula — Systems Thinking Effectiveness (STE)

```
STE = (Holism × Connection_Density × Temporal_Depth × Leverage_Accuracy) / (Reductionism × Silo_Walls × Short_Termism)

Where:
- Holism = % of decisions considering whole system (0-10)
- Connection_Density = Avg connections identified per entity (0-10)
- Temporal_Depth = Time horizon considered (years, normalized 0-10)
- Leverage_Accuracy = % interventions at high-leverage points (0-10)
- Reductionism = Tendency to break down vs. see whole (inverse, 0-10)
- Silo_Walls = Barriers between domains/departments (inverse, 0-10)
- Short_Termism = Focus on immediate vs. long-term (inverse, 0-10)

Target STE: Personal > 5.0 | Manager > 7.0 | Executive > 8.5 | CEO > 9.0
```
